- [X-UI](x-ui/README.md)

  ```bash
  wget https://raw.githubusercontent.com/vaxilu/x-ui/master/install.sh
  chmod +x install.sh && ./install.sh
  ```

- [Xray-script](Xray-script/README.md) (deprecated)

  ```bash

  ```
